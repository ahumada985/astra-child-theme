- **Tema Padre:** Astra

- **Sitio Web:** https://www.puppiesvet.cl

## 🚀 Instalación con WP Pusher

1. En WordPress Admin → WP Pusher → Install Theme
2. Repository: `ahumada985/astra-child-theme`
3. Branch: `main`
4. Activar "Push-to-Deploy"
